# Detecção de desvio de modelo

Com o recurso de detecção de desvio de modelo do Mix Modeler, as alterações no desempenho do modelo são detectadas automaticamente e os usuários são notificados. Eles podem tomar medidas imediatas para retornar o modelo ao desempenho ideal. Para obter mais informações, consulte [Descompasso do modelo](https://experienceleague.adobe.com/en/docs/mix-modeler/using/models/insights#model-drift){target="_blank"} na documentação do Mix Modeler.
